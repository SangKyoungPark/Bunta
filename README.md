# BUNTA - 베트남 3대 쌀국수 전문점

분타(BUNTA) 공식 웹사이트 리디자인 프로젝트

## 프로젝트 구조

```
Bunta/
├── index.html          # 메인 HTML
├── css/
│   └── style.css       # 스타일시트
├── js/
│   └── main.js         # 스크립트
└── README.md
```

## 섹션 구성

| 섹션 | 설명 |
|------|------|
| Hero | 풀스크린 배경 + 패럴랙스 스크롤 |
| Brand Story | 브랜드 스토리 + 핵심 수치 (10Y / 3.3.9 / 37%) |
| Key Numbers | 10Y, 3.3.9 비법, 6Day, 37% 강조 카드 |
| Menu | Bento Grid 레이아웃 (쌀국수/분짜/반쎄오/볶음밥/사이드) |
| Philosophy | 3가지 약속 (신선한 육수 / 엄선된 원재료 / 정직한 한 그릇) |
| Franchise | 가맹안내 (매출보증제 / 체계적 교육 / 상권 보호) |
| CTA | 창업 문의 + 공식 사이트 링크 |
| Footer | 메뉴 / 브랜드 / 가맹 링크 + SNS |

## 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - CSS Variables, Flexbox, Grid, 애니메이션
- **Vanilla JS** - IntersectionObserver, 패럴랙스
- **Google Fonts** - Noto Sans KR, Playfair Display
- **Unsplash** - 무료 이미지 (외부 CDN)

## 디자인 특징

- 2026 트렌드: Bento Grid, 미니멀리즘, 스크롤 애니메이션
- 베트남풍 골든 옐로우 컬러 테마 (`#E6A800`, `#FFD000`)
- 반응형 디자인 (데스크톱 / 태블릿 / 모바일)
- 스크롤 시 네비게이션 투명 → 화이트 전환
- IntersectionObserver 기반 스크롤 리빌 효과

## 핵심 숫자 (3.3.9 비법)

- **3** - 3종 육수
- **3** - 3종 고명
- **9** - 9가지 토핑
